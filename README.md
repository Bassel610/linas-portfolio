## 🔍 Overview

A modern portfolio website showcasing interior design projects with rich galleries, smooth carousels, and responsive layouts. Built with React + Vite for blazing-fast performance and Material-UI for a polished, professional look.

## ✨ Features

- 🖼️ **Project galleries** with responsive grids and image lightboxes
- 🎠 **Smooth carousels** powered by Swiper
- ✉️ **Contact form** with client-side validation
- 🎨 **Adaptive theming** — `colorthief` extracts dominant colors from images
- 🦥 **Lazy-loaded images** via `react-lazy-load-image-component` for fast initial paint
- 📱 **Mobile-first responsive design** across all breakpoints
- 🧭 **Client-side routing** with React Router 7
- 🔍 **SEO optimization** with semantic markup and meta tags

## 🛠️ Tech Stack

**Core**
- React 18.3
- Vite 5 (build tool)
- React Router 7

**UI & Styling**
- Material-UI 6 (`@mui/material`, `@mui/icons-material`)
- Emotion (CSS-in-JS)
- Swiper 11 for carousels

**Performance**
- `react-lazy-load-image-component` for image lazy-loading
- `colorthief` / `color-thief-browser` / `fast-average-color` for image color extraction

**Networking**
- Axios for HTTP requests

## 🚀 Getting Started

### Prerequisites

- Node.js ≥ 18
- npm ≥ 9

### Installation

```bash
git clone https://github.com/Bassel610/linas-portfolio.git
cd linas-portfolio
npm install
```

### Development

```bash
npm run dev          # Start Vite dev server (http://localhost:5173)
```

### Production Build

```bash
npm run build        # Create an optimized production build
npm run preview      # Preview the production build locally
```
